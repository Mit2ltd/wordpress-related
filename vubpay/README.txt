== Description ==

Create, process and manage VUB eCard payments for http://polymer.sav.sk

== Installation ==

= Using FTP =

1. Download `vubpay.zip`
2. Extract the `vubpay` directory to your computer
3. Upload the `vubpay` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
